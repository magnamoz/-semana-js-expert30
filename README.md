## :zap: Comandos para executar

```bash
# Clonar o repositório
$ git clone https://github.com/magnamoz/-semana-js-expert30

#Instalar as dependências
$ npm i

#Iniciar a aplicação no server
$ npm run dev

#Iniciar a aplicação no client 
(user 01 a user04)
$ npm run user01
```
